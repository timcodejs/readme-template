## 프로젝트 명

## Description
#### 해당 프로젝트는 CRA로 빌드된 프로젝트입니다. 
* recoil로 전역 상태 관리를 하며, react-query를 사용하여 서버와 통신합니다.
* react-hook-form과 react-select를 사용하여 Form Validation을 구현하였습니다.

## 실행 방법
dev (개발환경 실행) :
```js
yarn start
```
storybook (컴포넌트 테스트) :
```
yarn storybook
```
